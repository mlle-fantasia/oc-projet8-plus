Module: Store

Ici l'objet store sert à modifier le localStorage. Dans ce projet, le localstorage représente la base de donnée. l'objet store effectue les actions CRUD Create, Read, Update et Delete

(static) Store(name, callback)

Crée un store dans le localstorage du client avec une collection de todo vide s'il n'est existe pas déjà. Store utilise des callbacks car il n'y a pas de base de donée mais si non il faudrait utiliser des appel ajax
