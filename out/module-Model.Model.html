read(query, callback)

Cherche et retourne une todo (ou plusieurs) trouvée(s) dans le store. Si vous ne passez pas de query passé en paramettre, la fonction retourne toutes les todo. Si vous passez une string un nombre ou un objet, la fonction retournera la todo qui matche avec votre query
Parameters:
Name Type Description
query string | number | object A query to match models against
callback function Le callback
Source:
  • model.js, line 56
Example
model.read(1, func); // retournera la todo avec l'id 1
model.read('1'); // idem
model.read({ foo: 'bar', hello: 'world' }); // retournera la todo avec 'foo' === 'bar' && 'hello' === 'world'
